幫檸子做波段個股健檢：

1. 讀 `03 Invest-Swing/波段庫存.md` 的現有持倉。
2. 確認要檢哪一支：若 $ARGUMENTS 有指定就用它；沒有就列出持倉清單問我選。
3. 該支需要的即時數據若庫存表沒有，先問我（我會看盤後貼給你）：
   - 籌碼面：三大法人/主力近日買賣超、融資融券、大戶持股比、近日成交量能
   - 技術面：現價、相對 60MA/季線位置、最近波段高低點、是否創高/跌破支撐
4. 召喚兩位顧問框架分析：
   - **籌碼面** → 用 `/taiwan-intraday-signals` 視角：籌碼是進貨還是出貨？散戶 vs 大戶方向？量價是否背離？
   - **技術面 + 倉位** → 用 `/mark-minervini-perspective` 視角：趨勢階段？該加碼/減碼/續抱/出場？停損點在哪？
5. 輸出結構：
   - 🟢/🟡/🔴 一句話總評
   - 籌碼面結論（2-3 點）
   - 技術面 + 倉位建議（含具體停損價）
   - 下一步動作
6. 把結論追加進 `波段庫存.md` 的「健檢紀錄」（標日期），並依此更新該股「狀態」與「備註」。

紀律提醒：停損先設好、虧損不攤平、獲利才加碼（Minervini）。
