整理檸子的碎片靈感收件夾：

1. 讀 `07 Quick Capture & Hub/每日碎片化靈感隨記.md` 裡所有未處理的碎片。
2. 逐條判斷類型並處理：
   - **發文靈感** → 加 `#發文素材`，若需要搬去 [[每日熱門話題摘要]] 或留原地皆可（帶標籤即會進靈感候選庫）
   - **股票/投資知識** → 加 `#股票知識`。屬於長期（選股/估值、ETF/資產配置、總經/地緣/產業）的一律加進 [[投資知識庫]] 最下面的「📥 原始筆記」區塊（不是「✅ 個人結論」——結論層只能由 `/投資深聊` 確認後升級），標日期。屬於「技術/籌碼/波段」的整理進 [[波段資訊庫]]（不分層，波段留白不強求結論）。「心理/紀律」類進投資知識庫「心理/紀律」區塊，不分層
     - **來源標記**：碎片文字裡若有「聽說／群組／朋友說／XX群」等字眼 → 該行前面加 `[👥他人]`；若有「自己覺得／自己判斷／我認為／自己觀點」等字眼 → 加 `[💭自己]`；都沒提到就不加標記。純粹照字面判斷，不要自己腦補來源，寧可不標
   - **生活/醫美心得** → 整理進 [[愛美與醫美心得]]「碎片心得」區塊（帶 #發文素材 若可成內容）。同樣套來源標記：有「聽說／群組／朋友說／XX群」等字眼 → 加 `[👥他人]`；有「自己覺得／自己判斷／親身」等字眼 → 加 `[💭自己]`；沒提到不標
   - **想學的技能** → 整理進 [[想學習的技能清單]]
   - **主持稿/直播稿**（某場活動實際拿到或自己準備的稿子、注意事項）→ 整理成獨立檔案存進 `01 Brand/主持資料庫/01_事前準備/`，檔名格式 `直播腳本_案名_YYYYMMDD.md` 或 `主持稿_案名_YYYYMMDD.md`（照 [[直播腳本_滿貫大亨_20260630]] 的格式），完整保留原始稿子內容，不要精簡或摘要掉
   - **客戶/合作廠商產品資料**（如 ploom 的產品教育內容——是為了主持「他們的」活動用的廠商資料，**不是她自己的技能/課程**）→ 同樣存進 `01 Brand/主持資料庫/01_事前準備/`，檔名 `客戶資料_廠商名_YYYYMMDD.md`，完整保留不摘要。**關鍵區分：這是別人給的、為特定工作準備的資料，不進課程筆記庫**
   - **她自己上的主持技巧課筆記**（跟主持工作直接相關的課，她本人去學的）→ 整理進 `01 Brand/主持資料庫/03_資料庫/`，格式照該資料夾 README 範本（課程名稱/講師/日期 → 核心觀念 → 可套用技巧 → 實戰驗證）
   - **她自己上的跨領域課程筆記**（演員課/NLP/表達/溝通等，非主持專用的自我成長課）→ 整理進 `06 Lifestyle & Self-Growth/課程筆記庫/`，**按應用場景分**（表達與台風/心理與溝通），不按課程名分，格式照該資料夾 README 範本
   - **香氛蠟燭配方/工作室知識**（香精配方、蠟種比例、成本、定價、上架、場地租借細節）→ 整理進 `01 Brand/香氛工作室/`，配方類進 `配方庫/`（格式照該資料夾 README 範本），成本/上架/場地類進對應子檔，完整保留數字與比例不摘要
   - **純雜念/待辦** → 轉成 `- [ ]` 待辦或刪除
3. 已處理的碎片，在 Inbox 該行前面標 `✅` 或移走（不要刪掉原始時間戳，保留軌跡）。
4. **重複候選檢查**（不自動刪，只提醒）：分流前，快速比對這條碎片的內容是否跟目標中樞（品牌中樞/COS經營/撥撥獺獺IP/投資知識庫等）裡已有的段落高度重疊。若疑似重複，不要默默整理掉，改成回報：「這條疑似跟 [[某中樞]] 的『某段落』重複，要我確認後清掉原始碎片嗎？」，等我回答才動手刪除或封存。
5. **投資知識庫衝突檢查**（長期類：選股/估值、ETF/資產配置、總經/地緣/產業）：這條新碎片跟 [[投資知識庫]]「📥 原始筆記」裡近期（約2週內）已有的內容，是不是講同一支股票/同一議題但方向相反（例：一則說看多、另一則說要停利看空）？矛盾**允許並存**、不用刪也不用選邊站，但要回報提醒：「這條跟投資知識庫 X/X 那則方向相反，都留著標矛盾，之後 `/投資深聊` 再一起判斷」。純粹提醒，不擅自決定誰對。
6. 回報：處理了幾條、各分到哪、有沒有無法判斷需要我確認的、有沒有偵測到重複候選、有沒有偵測到投資知識衝突候選。

原則：Inbox 只負責「3 秒捕獲」，分類由這個指令做，不要讓我手動整理。刪除/封存一律先問過我。
