---
description: 建立一個新工作流/專案節點並自動串連到神經網路
argument-hint: [工作流名稱]
---

幫我把「$ARGUMENTS」加進第二大腦：

1. 先問我（若我還沒講）：這屬於哪個腦區（品牌/投資/波段/其他）？狀態是什麼（planning/active/beta/review）？
2. 在 `04 Projects/` 建一個 `$ARGUMENTS.md`，含 frontmatter：
   ```
   type: project
   area: <腦區>
   status: <狀態>
   updated: <今天日期>
   ```
3. 內容骨架：標題、現況、連結（`所屬：[[對應中樞]]`）、待辦、紀錄。
4. 到對應的「○○中樞.md」的連結區，加一條 `[[$ARGUMENTS]]`，讓神經網路接上。
5. 回報：建了什麼檔、接到哪個中樞。提醒我回 Obsidian 重開關係圖譜看新節點。

`Home` 的專案表格會自動更新，不用動它。
