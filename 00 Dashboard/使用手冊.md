## 這整套東西是什麼

我把生活跟工作的所有資料都放進 Obsidian（一個資料夾裡全是 Markdown 筆記），然後用 Claude Code 當我的「神經系統」——我打字告訴它要做什麼，它就幫我讀寫這些筆記、整理資料、生成內容、跟 Discord bot 互動。

我不用自己手動歸檔，丟進去的碎片會自動被分類。

## 我每天會打開的東西

**Home（今日主控）** 是我每天的第一站。打開後看到：
- 今日待辦、今日行程、靈感碎片數、本週待辦——四張小卡一眼掃過
- 行事曆（自動抓 iCloud 60 天內的事件）
- 每日校準：習慣打卡 + 發文數據回填（兩個 tab 切換）
- 未來 14 天發文計畫
- 靈感碎片輸入框 + 靈感圖書館（連到真正有用的知識，不是流水帳）

其他像「本週目標」「工作流」「個人資產」這些不常變的東西，我收進最下面「📥 每週才看」那個收合區，點開才看到，平常不會擋住畫面。

**發文相關**：
- 我想到什麼靈感，直接丟進 `07 Quick Capture & Hub/每日碎片化靈感隨記.md`（或 Discord #快速捕獲），不用分類，3 秒打完就好
- 想到「這個可以發文」就標 `[發文素材]`，之後在 Discord 打 `/整理` 會自動分類
- 打 `/發文日常` 會幫我看今天該發什麼、有沒有數據要回填、靈感池裡有什麼可以用

## 我想找「真正有用的資料」去哪裡

Home 頁面右邊的「📚 靈感圖書館」只放蒸餾過的東西，不是原始筆記流水帳：
- **發文靈感池**：已經整理過、可以直接拿去發文的素材（社群熱點/熱門話題/時事梗，**股市新聞不算在內**）
- **股市新聞**（2026-07-08 新增，側邊欄「工具」）：Brief 裡的股市新聞獨立拆出來，純閱讀盤前資訊用，不會混進發文靈感
- **概念卡**：我從實戰裡蒸餾出的心法（像「demo 要大於宣稱」「90% 法則」這種可以套用在很多地方的判斷原則），放在 `08 Skills 方法庫/概念卡/`
- **品牌中樞當前重點**：我現在的定位、策略、顧問建議的蒸餾結論
- **撥撥獺獺 IP**：角色設定+內容策略的完整文件
- **個股總覽**：股票持倉/估值/模型/研究一次看完，不用開好幾個檔案拼
- **接案執行庫**：報價規律+實戰紀錄

如果我要的是「原始沒整理的碎片」，那不在這裡，去側邊欄「快速捕獲」找。

## 常見問題

**Q：Discord 突然一直出現同一個錯誤訊息？**
通常是 bot 需要重啟，或 Claude CLI 登入過期。跟 Claude 說「重啟 bot」讓它幫我處理。

**Q：改完東西後畫面看起來沒變化？**
Obsidian 不會自動重新整理已經開著的頁面。切到別的分頁再切回來，或按 `Ctrl+P` 找「Reload app without saving」。

**Q：某個資料看起來重複很多次？**
說一聲要「資料脫水」，會幫我篩掉重複/過期的內容，但刪除前一定會跟我確認再動手，不會自己默默刪掉東西。
