# 大盤日報 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 新增一支每日兩次自動推播的「大盤日報」（早盤開盤前功課 + 午盤收盤後總結），數據全部來自真實免費 API（TWSE、Yahoo Finance、TAIFEX），AI 只負責用真實數字寫短評，不編造任何數據。

**Architecture:** 新建 `market-daily-report.mjs`（跟 `holdings-price-updater.mjs`/`etf-monitor.mjs` 同一種單檔模組風格：純函式抓資料 + 組報告 + export 給 `discord-bot.mjs` 呼叫），在 `discord-bot.mjs` 裡加兩個排程時間點 + 一個手動指令 `/大盤`。報告內容累積寫進 `03 Invest-Swing/大盤日報.md`。

**Tech Stack:** Node.js（ES modules，跟現有 brain-bot 一致）、內建 `https` 模組、`iconv-lite`（解析 TAIFEX 的 Big5 編碼 CSV）、Gemini `gemini-flash-lite-latest` API（跟 `news-brief-free.mjs` 同一顆模型，0 Claude 額度）。

## Global Constraints
- 這個專案沒有測試框架（`package.json` 只有 `discord.js` 一個 dependency），每個函式的驗證方式是**直接用 `node -e` 或檔案自帶的 CLI 執行區塊手動跑一次，檢查真實輸出**，跟這個 repo 裡其他腳本（`holdings-price-updater.mjs`、`etf-monitor.mjs`、`earnings-reviewer.mjs`）的驗證方式一致。
- 所有對外部 API 的請求都要帶 `User-Agent: Mozilla/5.0`（TWSE/TAIFEX 有些端點沒帶會被拒絕，這是現有腳本已經驗證過的坑）。
- 抓不到的數據一律回傳 `null`（不是丟例外、不是編造數字），呼叫端把 `null` 轉成人看得懂的「⚠️ 今日未能取得」，絕對不能讓 AI 自己補數字。
- 時間一律用 `Asia/Taipei` 時區換算（跟現有 `holdings-price-updater.mjs` 的 `taipeiToday()` 做法一致）。
- Gemini 摘要 prompt 必須明確禁止引用「提供數據以外的任何數字或臆測」，這是使用者這次需求的核心（不能AI亂掰）。
- `VAULT` 路徑常數沿用 `discord-bot.mjs` 現有的 `const VAULT = env.VAULT_PATH || "C:\\tia\\vault";`；`GEMINI_KEY` 沿用 `const GEMINI_KEY = env.GEMINI_API_KEY || "";`。

```javascript
// ---- 共用：HTTP 請求（回傳原始 Buffer，呼叫端自己決定編碼）----
function httpsGetBuffer(url, extraHeaders = {}) {
  return new Promise((resolve) => {
    https.get(url, { headers: { 'User-Agent': 'Mozilla/5.0', ...extraHeaders } }, (res) => {
      const chunks = [];
      res.on('data', (c) => chunks.push(c));
      res.on('end', () => resolve(Buffer.concat(chunks)));
    }).on('error', () => resolve(null));
  });
}

function httpsPostBuffer(url, bodyStr, extraHeaders = {}) {
  return new Promise((resolve) => {
    const u = new URL(url);
    const req = https.request({
      hostname: u.hostname, path: u.pathname + u.search, method: 'POST',
      headers: {
        'User-Agent': 'Mozilla/5.0',
        'Content-Type': 'application/x-www-form-urlencoded',
        'Content-Length': Buffer.byteLength(bodyStr),
        ...extraHeaders,
      },
    }, (res) => {
      const chunks = [];
      res.on('data', (c) => chunks.push(c));
      res.on('end', () => resolve(Buffer.concat(chunks)));
    });
    req.on('error', () => resolve(null));
    req.write(bodyStr);
    req.end();
  });
}

async function httpsGetJSON(url) {
  const buf = await httpsGetBuffer(url);
  if (!buf) return null;
  try { return JSON.parse(buf.toString('utf8')); } catch { return null; }
}
```

```javascript
async function fetchMarginTrading(dateCompact) {
  const url = `https://www.twse.com.tw/rwd/zh/marginTrading/MI_MARGN?response=json&date=${dateCompact}&selectType=ALL`;

  const financeRow = rows.find((r) => r[0] === '融資(交易單位)');
  const shortRow = rows.find((r) => r[0] === '融券(交易單位)');
  if (!financeRow || !shortRow) return null;
  const num = (s) => parseFloat((s || '').replace(/,/g, ''));

  const shortBalance = num(shortRow[5]);
  const shortPrev = num(shortRow[4]);
  if (isNaN(financeBalance) || isNaN(shortBalance)) return null;
  return {
    financeBalance,
    financeChange: isNaN(financePrev) ? null : financeBalance - financePrev,
```

```javascript
async function fetchOneUSIndex(symbolEncoded) {
  const url = `https://query1.finance.yahoo.com/v8/finance/chart/${symbolEncoded}`;
  const j = await httpsGetJSON(url);
  const meta = j?.chart?.result?.[0]?.meta;
  if (!meta || typeof meta.regularMarketPrice !== 'number' || typeof meta.previousClose !== 'number') return null;
  const price = meta.regularMarketPrice;
  const prev = meta.previousClose;
  const change = price - prev;
  const changePct = (change / prev) * 100;
  return { price: Math.round(price * 100) / 100, change: Math.round(change * 100) / 100, changePct: Math.round(changePct * 100) / 100 };
}
async function fetchUSIndices() {
  const [dow, nasdaq, sp500] = await Promise.all([
    fetchOneUSIndex('%5EDJI'),
    fetchOneUSIndex('%5EIXIC'),
    fetchOneUSIndex('%5EGSPC'),
  ]);

}
```

### Task 6: TAIFEX 台指期夜盤抓取

**Files:**

**Interfaces:**

- Produces：`fetchNightFutures(dateSlash)` → `Promise<{close:number, changePt:number, changePct:number, contractMonth:string} | null>`（抓當天「盤後」時段、最近月份合約的收盤資訊）

```javascript
// ---- TAIFEX 台指期夜盤（盤後時段）----
// 官方表單查詢端點是 POST + Big5 編碼 CSV，欄位順序：
// 交易日期,契約,到期月份(週別),開盤價,最高價,最低價,收盤價,漲跌價,漲跌%,成交量,結算價,未沖銷契約數,
// 最後最佳買價,最後最佳賣價,歷史最高價,歷史最低價,是否因訊息面暫停交易,交易時段,價差對單式委託成交量
async function fetchNightFutures(dateSlash) {
  const body = `down_type=1&commodity_id=TX&commodity_id2=&queryStartDate=${encodeURIComponent(dateSlash)}&queryEndDate=${encodeURIComponent(dateSlash)}`;
  const buf = await httpsPostBuffer('https://www.taifex.com.tw/cht/3/futDataDown', body);
  if (!buf || buf.length === 0) return null;
  const text = iconv.decode(buf, 'big5');
  const lines = text.split('\n').map((l) => l.trim()).filter(Boolean);
  if (lines.length < 2) return null;
  // 找最近月份合約的「盤後」列：先把所有 TX 的盤後列挑出來，用到期月份字串排序取最小（最近月）
  const nightRows = lines.slice(1)
    .map((l) => l.split(','))
    .filter((cols) => cols[1] === 'TX' && cols[17] === '盤後');
  if (nightRows.length === 0) return null;
  nightRows.sort((a, b) => a[2].trim().localeCompare(b[2].trim()));
  const row = nightRows[0];
  const close = parseFloat((row[6] || '').replace(/,/g, ''));
  const changePt = parseFloat((row[7] || '').replace(/,/g, ''));
  const changePctStr = (row[8] || '').replace('%', '').trim();
  const changePct = parseFloat(changePctStr);
  if (isNaN(close)) return null;
  return {
    close,
    changePt: isNaN(changePt) ? null : changePt,
    changePct: isNaN(changePct) ? null : changePct,
```

```javascript
// ---- Gemini 情境摘要：只能就給定的真實數字寫短評，禁止引用其他數據或臆測未來 ----
async function summarizeWithGemini(factsText, apiKey) {
  if (!apiKey) return '（未設定 GEMINI_API_KEY，略過情境摘要）';
  const prompt = `你是台灣波段投資顧問，語氣中性、給情境分析、不下指令（不能說「該買該賣」）。

${factsText}

  const url = `https://generativelanguage.googleapis.com/v1beta/models/gemini-flash-lite-latest:generateContent?key=${apiKey}`;

  for (let attempt = 1; attempt <= 3; attempt++) {
    try {
      const r = await fetch(url, { method: 'POST', headers: { 'Content-Type': 'application/json' }, body });
      const j = await r.json();
      if (r.ok) return (j.candidates?.[0]?.content?.parts?.[0]?.text || '').trim() || '（今日無情境摘要）';
      if (r.status === 503 && attempt < 3) { await new Promise((res) => setTimeout(res, attempt * 10000)); continue; }
      return `（情境摘要生成失敗：Gemini ${r.status}）`;
    } catch (e) {
      if (attempt === 3) return `（情境摘要生成失敗：${e.message}）`;
    }
  }
```

```javascript
// ---- 格式化小工具 ----
function fmtPct(n) { return n === null || n === undefined || isNaN(n) ? '⚠️未能取得' : `${n >= 0 ? '+' : ''}${n.toFixed(2)}%`; }
function fmtNum(n) { return n === null || n === undefined || isNaN(n) ? '⚠️未能取得' : n.toLocaleString('en-US'); }
function fmtSigned(n, unit = '') { return n === null || n === undefined || isNaN(n) ? '⚠️未能取得' : `${n >= 0 ? '+' : ''}${n.toLocaleString('en-US')}${unit}`; }

function ensureReportFile(vault) {
  const path = join(vault, '03 Invest-Swing', '大盤日報.md');
  if (!existsSync(path)) {
    writeFileSync(path, '# 大盤日報\n\n> 每日開盤前+收盤後大方向總覽，數據來自 TWSE/Yahoo Finance/TAIFEX 官方公開資料，AI 僅就真實數字做情境描述。\n', 'utf8');
  }
  return path;
}
```
