# 每日熱門話題摘要（Daily Brief）

> 每天凌晨 AI 自動抓取 → 過濾 → 生成摘要頁面。
> 每份 Brief 必須帶標籤 `#發文素材`，讓 [[社群自動發文]] Skill 可撈取。

---

## 自動化說明

**觸發時間**：每日凌晨（建議 01:00-03:00）
**抓取來源**：台灣 FB / Threads 趨勢 + 即時新聞熱詞
**生成格式**：每日一份獨立筆記，命名為 `Brief YYYY-MM-DD`

---

## Brief 日誌索引

```dataviewjs
const pages = dv.pages('"01 Brand"')
  .where(p => p.file.name.startsWith("Brief "))
  .sort(p => p.file.name, "desc");

dv.table(
  ["日期", "話題摘要", "股市摘要", "已使用"],
  pages.map(p => [
    p.file.link,
    p["話題摘要"] || "—",
    p["股市摘要"] || "—",
    p["已使用"] || "⬜"
  ])
);
```

---

## 每份 Brief 標準格式

由 `gemini-brief.mjs` 自動產生（Gemini API + Google 搜尋 grounding，強制真實來源URL），格式：

```markdown
---
type: brief
date: YYYY-MM-DD
tags: [發文素材]
話題摘要: <影片梗+熱門話題 topic 清單，頓號分隔>
股市摘要: <股市新聞 headline 清單，頓號分隔>
已使用: ⬜
來源: Gemini 自動生成（僅收錄有來源 URL 的項目）
---

# Brief YYYY-MM-DD

## 🎬 熱門影片梗
## 📌 股市今日重點總結
## 📈 股市重要新聞
## 🔥 熱門話題（社群切入角度）
## 🔍 搜尋紀錄
```

**產生方式（單一邏輯，2026-07-11 起統一）**：每天 02:00 Discord bot 自動排程 `generateBrief()`；需要立即手動重跑一次時用 `/生成Brief`（Claude Code 指令，直接呼叫同一支 `gemini-brief.mjs`，0 Claude 額度）。不再有第二套獨立生成邏輯。
