## 現行最優打法

### FB 個人（最強擴散場）
1. **Mode C 炎上/立場/脆弱文**（無圖也爆）→ 非粉觸及 46-85%，觸及量 1940-12000
   - 觸發條件：COS圈事件、社會觀察、個人立場宣言、情緒共鳴
   - 無圖長文不需等素材，有話題就插隊
2. **帶圖工作照 F2（強主題）**→ 非粉 47-66%，觸及 1137-1656
   - 強主題 = COS造型、SG工作、有張力的場景（車展、護士主題）
   - 普通日常照效果下降（比基尼廢文非粉 15%，假髮閒魚 15-21%）
3. **Mode A 帶爆點圖**（非廢文）→ 非粉最高 61%（滿貫預告帶遊戲圖）
   - 有「事件感」的圖（活動/遊戲/強主題）> 純日常圖
4. **純廢文/日常碎念**→ 15-21% 非粉，觸及 337-1547，**不擴散**，僅鐵粉深化用

### Gracie 粉專（推薦算法已啟動，穩定擴散）
1. **F2 工作照 + 新鮮場景**→ 非粉 47-58%，觸及 1063-2065（最強：台中新車大展 58%）
   - 車展、游泳池SG、大場活動 > 試妝 > 純COS（試妝 23%，確認題材天花板低）
2. **F12 預告/聲播**→ 非粉 49-59%，觸及 921-2007（最強：滿貫大亨預告 163讚 2007觸及）
   - F12 是 Gracie 最強公式，配合活動預告威力最大
3. **封面無本人照 = 限流**，AI純內容觸及崩壞（試妝文限流實證）

### 個人 IG（深化型，非擴散工具）
- 非粉觸及長期 3-8%，主要觸及鐵粉（98.6% 追蹤者）
- 功能：作品集、業配、廠商配合文的展示場
- 不以非粉觸及評估，看留存率和收藏數

### Threads（高變異，有公式依賴）
- **F7 求討論/帶情境**→ 最強：瀏覽 3740，留言 27（蕁麻疹求討論），82讚（台中展場聯動）
- **F7 純廢吐**→ 瀏覽 59-538，互動低，無明顯擴散
- **規律：跟 FB 主帳同天有話題性內容時，Threads 同步效果翻倍**
  - 6/30 Mode B（觀看1709）+ Threads F7（82讚7留）—— 兩篇聯動效應明顯
- **COS Threads**→ 幾乎無觸及，演算法推不出 COS 圈外

### 撥撥 IG + TikTok
- **TK 觸及天花板 >> IG**（同素材：TK 550-638 vs IG 47-234）
- 略過率主因：①前1秒沒動作 ②**題材普世度**（料理59% vs 遊戲84.8%）
- 改善路線：普世題材（食物/起床/可愛動作）> 遊戲/特定IP題材

### 逆水寒官方社團 R30（隱藏王牌）
- 潮光COS cross-post：337讚 35留 2分享，**超過任何主帳單篇**
- 遊戲/逆水寒 COS 相關文，主帳發完必 cross，成本 0，ROI 最高

## 最新發現
<!-- 每次分析的新增洞察，標日期 -->

### 2026-07-03（第二次分析，資料 6/5-6/30）

**1. Mode C 無圖長文是 FB 個人最強武器，但配額沒用滿**
炎上/立場/脆弱類 Mode C：每篇都突破非粉 46% 以上，最高 85%，但這輪只發了 5 篇。
廢文 Mode A 塞太多（比基尼、假髮、閒魚、進廠維修），這些非粉觸及都在 15-21%。
→ **下輪 Mode C 比例要提高，Mode A 要有「事件感」才排**

**2. 試妝 COS 系列（6/25）確認擴散力弱**
6 個平台同步發，Gracie粉專 23%，COS 粉專 23%，COS IG 29%——均偏低。
試妝題材對非 COS 圈受眾吸引力有限。**COS 線維持存在感即可，不期待擴散**。

**3. Gracie 粉專：工作+新鮮場景 >> 試妝 COS**
台中新車大展（6/29）58% 非粉觀看 2065，信義富境SG（6/21）47%——都強。
試妝（23%）差距近 2.5 倍，確認**新鮮工作場景 > COS試妝**，Gracie 以工作照為主力。

**4. Threads F7 效果高度依賴「話題性 + FB聯動」**
同天有 FB 話題文時，Threads 同步的效果翻倍（6/30 範例：82讚7留1分享）。
純 Threads 即興廢吐（維琳娜假髮、閒魚）幾乎 0 觸及。
→ **Threads 不排即興廢文，要跟當天 FB 強主題文連動，或等有話題感的 F7**

**5. 撥撥「題材普世度」比「前1秒動作」更決定略過率**
料理翻車（前1秒撥撥煮菜）= 略過率 59%。
三角洲奪舍（前1秒有動作）= 略過率 84.8%。
遊戲題材對非遊戲受眾普世度低，略過率天然高。
→ **撥撥下一批：食物/起床/自然動作 > 遊戲/特定IP**
