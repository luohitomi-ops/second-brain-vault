## 現行最優打法
<!-- /爆款優化 每次更新這裡，越來越精準 -->

### FB 個人（最強擴散場，但 7/1-7/8 這輪 Mode C 配額完全沒用滿）
1. **Mode C 炎上/立場/脆弱文**（無圖也爆）→ 非粉觸及 46-85%，觸及量 1940-12000
   - 觸發條件：COS圈事件、社會觀察、個人立場宣言、情緒共鳴
   - 無圖長文不需等素材，有話題就插隊
2. **帶圖工作照 F2（強主題）**→ 非粉 47-66%，觸及 1137-1656
   - 強主題 = COS造型、SG工作、有張力的場景（車展、護士主題）
   - 普通日常照效果下降（比基尼廢文非粉 15%，假髮閒魚 15-21%）
3. **Mode A 帶爆點圖**（非廢文）→ 非粉最高 61%（滿貫預告帶遊戲圖）
   - 有「事件感」的圖（活動/遊戲/強主題）> 純日常圖
4. **純廢文/日常碎念**→ 15-21% 非粉，觸及 337-1547，**不擴散**，僅鐵粉深化用
5. **⚠️ 7/1-7/8 這輪只發了 Mode A 工作文（74-104讚）+ 1 篇 Mode C 小情緒文**（工作滿檔壓力，僅33讚/觀看352，遠低於 Mode C 歷史峰值），**沒有真正炎上/立場文可驗證** → reach 表現明顯低於 Mode C 全開時期，下一輪要主動插隊 1-2 篇

### Gracie 粉專（推薦算法已啟動；7/1-7/3 出現粉專史上首次 mega-viral 突破）
1. **🏆 大型公開活動/展覽 tag（紅人展/成人展/車展）= reach 催化劑，等級遠超日常工作照**
   → 非粉 90-91%，觸及 4.8萬-5.9萬（7/1 展場工作照+廠商補貼、7/3 TRE紅人展SG拍照場景 兩篇實證，是粉專歷史前高的 25-29 倍）
   - 機制推測：大型公眾展覽自帶搜尋/話題熱度，貼文掛上展覽名稱/場景就能吃到展覽本身的公眾關注度，不只靠粉絲圈內擴散
   - **史上最強單篇公式（7/1 實證）**：「職場遭遇（工作被取消/委屈故事）+ 下一步預告」→ 1164讚 61留 25分享
   - 互動率反而較低（2.1-2.9%，遠低於 F12 常態的 6-8%）→ 屬**擴散型**貼文（拉新為主），跟深化型公式不能直接比讚數
   - 樣本僅 2 篇，屬新發現，需下次遇到同類大型展覽再驗證是否可複製
2. **F2 工作照 + 新鮮場景（非大型公眾展覽）**→ 非粉 47-66%，觸及 1063-3676
   - 車展、SG拍照場景、大場活動 > 試妝 > 純COS（試妝 23%，確認題材天花板低）
3. **F12 預告/聲播**→ 非粉 49-59%，觸及 921-2968；**互動率穩定 6-8%（7/5、7/8 實證），是最可靠的「深化型」公式**，轉化既有粉絲效率比 mega 展覽文更高
4. **⚠️ 業配類文字稿（非工作照/非活動現場）表現弱**→ 7/6 業配文章 僅 11讚/觀看557，遠低於任何 F2/F12，**業配文字稿只做交差用，不能預期帶動 reach 成長**
5. **封面無本人照 = 限流**，AI純內容觸及崩壞（試妝文限流實證）

### 個人 IG（深化型，非擴散工具；🔴 2026-07-10 確立「TA 女粉化」為指導方向，蓋過短期 reach 指標）
- 非粉觸及長期 3-8%，主要觸及鐵粉（98.6% 追蹤者）
- 功能：作品集、業配、廠商配合文的展示場
- **評估標準改變（2026-07-10 用戶明確指示）：個人IG 不追求馬上衝觸及，改用「選題+語氣是否對女性受眾」為優先判準**，觸及/讚數退居次要參考，看留存率、收藏數、（有數據時的）性別比例
- **受眾性別比例**（目前唯一 1 篇數據）：7/3「台中展場照+植髮心得」文 女性受眾 55.3%
  - 該文特徵：工作照 + 美容/自我提升心得（植髮）混合敘事，不是純外貌/身材照
  - → **選題方向**：工作照＋美容心得／自我提升／專業知識混合敘事，優先於純外貌/身材/性感向照片
  - → **語氣方向**：分享型、心得型、relatable（像在跟朋友聊保養/成長），避免純展示/邀觀看型 caption
- 樣本仍僅 1 篇，下一輪起持續排此類型貼文累積驗證，**即使單篇觸及不突出也不算失敗**——只要選題語氣方向對，就照 standing rule 執行，不用每篇都拿 reach 檢討
- **⚠️ 業配同步文表現弱**：7/6 業配文章同步IG版 僅 3讚/觀看185，業配轉發型內容不預期帶動 reach，純服務交差用

### Threads（高變異，有公式依賴）
- **F7 求討論/帶情境**→ 最強：瀏覽 3740，留言 27（蕁麻疹求討論），82讚（台中展場聯動）
- **F7 純廢吐**→ 瀏覽 59-538，互動低，無明顯擴散
- **規律：跟 FB 主帳同天有話題性內容時，Threads 同步效果翻倍**
  - 6/30 Mode B（觀看1709）+ Threads F7（82讚7留）—— 兩篇聯動效應明顯
  - 7/4、7/7 TRE紅人展聯動短版各穩定 75讚3留，確立「事件聯動短版」有 70-80讚 的可靠地板，非單次僥倖
- **COS Threads 已於 2026-07-06 停更、併入個人 @yuzukiluo**（歷史觸及幾乎全 0，此條保留做歷史紀錄用）

### 撥撥 IG + TikTok
- **TK 觸及天花板 >> IG**（同素材：TK 550-638 vs IG 47-234）
- 略過率主因：①前1秒沒動作 ②**題材普世度**（料理59% vs 遊戲84.8%）
- 改善路線：普世題材（食物/起床/可愛動作）> 遊戲/特定IP題材

### 逆水寒官方社團 R30（隱藏王牌）
- 潮光COS cross-post：337讚 35留 2分享，**超過任何主帳單篇**
- 遊戲/逆水寒 COS 相關文，主帳發完必 cross，成本 0，ROI 最高

## 最新發現
<!-- 每次分析的新增洞察，標日期 -->

### 2026-07-10（第三次分析，資料 7/1-7/8）

**1. 🏆 Gracie 粉專出現史上首次 mega-viral，主因是「大型公開展覽 tag」不是公式本身**
7/1（展場工作照+廠商補貼）1164讚/61留/25分享/觸及5.9萬/非粉91%，7/3（TRE紅人展SG拍照場景）1335讚/32留/34分享/觸及4.8萬/非粉90%——兩篇都是粉專史上前高的 25-29 倍。同期非展覽的一般 F2/F12（成人展主力照、聲播預告）仍是常態 100-238讚。差異變數是「掛上大型公眾展覽名稱/場景」，推測吃到展覽本身的公眾搜尋/話題熱度，不是純靠粉絲圈擴散。
→ **下次遇到車展/紅人展/成人展等大型公開活動，優先排 Gracie 粉專工作照+故事型貼文，reach 天花板遠高於日常**

**2. FB 個人這輪 Mode C 炎上/立場文配額再度掛零**
7/1-7/8 只發 Mode A 工作文（74-104讚）+ 1 篇 Mode C 小情緒文（工作滿檔壓力，僅33讚/觀看352），沒有真正炎上/立場文。呼應 7/3 分析已提過的問題，這輪仍未改善。
→ **連續兩輪都沒用滿 Mode C 配額，是目前 FB 個人 reach 沒到頂的主因，下一輪必插隊**

**3. 業配轉發型內容（文字稿/同步IG版）在兩個平台都是本輪最弱項**
7/6 業配文章 Gracie粉專 11讚/觀看557、個人IG 3讚/觀看185，遠低於任何工作照/活動文。
→ **業配文字稿只做交差用，不能用來評估帳號健康度，排程時心理預期要降低**

**4. 個人 IG 首次拿到受眾性別數據，呼應「TA 女粉化」新方向**
7/3「台中展場照+植髮心得」文女性受眾 55.3%，是唯一有性別拆分的貼文，特徵是工作照+美容心得混合敘事而非純外貌照。對應品牌中樞新確立的 standing rule（TA 慢慢往女粉靠攏）。
→ **樣本僅 1 篇，下一輪起主動排美容/心得/專業向混合貼文，累積更多性別數據做驗證**

**5. Threads 事件聯動短版確立可靠地板**
7/4、7/7 TRE紅人展聯動短版各 75讚/3留，跟 6/30 台中展場（82讚）同量級，證明「FB/粉專有話題文時同步 Threads 短版」不是單次僥倖，是可重複的 70-80讚 地板。

**6. 🔴 個人IG 評估標準確立轉向：從「衝觸及」改成「選題語氣對女性受眾」**
用戶明確指示：今後個人IG 不追求馬上爆觸及，選題和語氣要優先對準逐漸女粉化的方向。這不是新數據，是**指導方針調整**——原本 7/3 植髮心得文 55.3% 女性受眾只是「單一觀察」，現在正式升格為**下一輪選題的預設優先序**（工作照+美容心得/自我提升混合敘事 > 純外貌照），且評估這類貼文時**不看短期 reach 是否亮眼，看選題語氣方向對不對**。
→ 下一輪個人IG 排程時，優先選「工作照+保養/自我提升/專業心得」類型題材，caption 語氣走分享/relatable，不是純展示邀讚。

### 2026-07-03（第二次分析，資料 6/5-6/30）

**1. Mode C 無圖長文是 FB 個人最強武器，但配額沒用滿**
炎上/立場/脆弱類 Mode C：每篇都突破非粉 46% 以上，最高 85%，但這輪只發了 5 篇。
廢文 Mode A 塞太多（比基尼、假髮、閒魚、進廠維修），這些非粉觸及都在 15-21%。
→ **下輪 Mode C 比例要提高，Mode A 要有「事件感」才排**

**2. 試妝 COS 系列（6/25）確認擴散力弱**
6 個平台同步發，Gracie粉專 23%，COS 粉專 23%，COS IG 29%——均偏低。
試妝題材對非 COS 圈受眾吸引力有限。**COS 線維持存在感即可，不期待擴散**。

**3. Gracie 粉專：工作+新鮮場景 >> 試妝 COS**
台中新車大展（6/29）58% 非粉觀看 2065，信義富境SG（6/21）47%——都強。
試妝（23%）差距近 2.5 倍，確認**新鮮工作場景 > COS試妝**，Gracie 以工作照為主力。

**4. Threads F7 效果高度依賴「話題性 + FB聯動」**
同天有 FB 話題文時，Threads 同步的效果翻倍（6/30 範例：82讚7留1分享）。
純 Threads 即興廢吐（維琳娜假髮、閒魚）幾乎 0 觸及。
→ **Threads 不排即興廢文，要跟當天 FB 強主題文連動，或等有話題感的 F7**

**5. 撥撥「題材普世度」比「前1秒動作」更決定略過率**
料理翻車（前1秒撥撥煮菜）= 略過率 59%。
三角洲奪舍（前1秒有動作）= 略過率 84.8%。
遊戲題材對非遊戲受眾普世度低，略過率天然高。
→ **撥撥下一批：食物/起床/自然動作 > 遊戲/特定IP**

## 待驗證假設
- 試妝題材（6/25 多篇）非粉觸及偏低 → ✅ 已驗證，題材本身擴散力確實弱
- Mode B 工作帖 + Threads 聯動是否有放大效應 → ✅ 已驗證（7/4、7/7 TRE聯動短版穩定 75讚，非單次僥倖）
- 個人 IG 非粉觸及是否可以透過 F2 強主題照提升 → 台中展場 8% 比醫美配合 3.8% 高，場景強度有影響，待累積
- 🆕 大型公開展覽 tag 是否為可複製的 Gracie 粉專 reach 催化劑 → 目前僅 2 篇實證（7/1、7/3），需下次遇到同類大型展覽（車展/紅人展/成人展）再驗證，暫不能當成穩定公式
- 🆕 「工作照+美容/心得混合敘事」是否能穩定拉高 IG 女性受眾比例 → 目前僅 1 篇有性別數據（55.3%），需累積 3-5 篇同類型貼文才能判斷
