# 香氛工作室

> 香氛蠟燭事業的知識中樞：配方、成本、上架資料統一放這裡。
> 對應品牌中樞「香氛工作室三用」策略（① 實品蠟燭銷售 ② 場地租借 ③ 直播空間）。

## 結構
`/整理` 會把捕獲頻道貼的香氛相關碎片，各自存成獨立檔案放在這個資料夾根目錄（檔名：`香氛_{標題}_{日期}.md`），目前不分子資料夾——量還少，之後配方/成本/通路資料多起來，再視情況手動整理成子分類。

## 配方記錄建議格式
```
## 蠟燭名稱 / 建立日期
### 香精配方（品牌+比例）
### 蠟種與比例（大豆蠟/棕櫚蠟/蜂蠟…）
### 燃燒測試（香擴散度/燃燒時間/隧道效應有無）
### 成本（單支材料成本 NT$）
### 調香心得 / 下次調整
```
