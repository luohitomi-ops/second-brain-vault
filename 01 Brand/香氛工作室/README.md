# 香氛工作室

> 香氛蠟燭事業的知識中樞：配方、成本、上架資料統一放這裡。
> 對應品牌中樞「香氛工作室三用」策略（① 實品蠟燭銷售 ② 場地租借 ③ 直播空間）。

## 結構（各類內容整理進對應子檔，`/整理` 會自動分流到這裡）

| 子項目 | 放什麼 |
|---|---|
| `配方庫/` | 每支蠟燭的香精配方、蠟種比例、燃燒測試結果、調香心得 |
| `成本與定價.md` | 材料成本、單支成本、定價邏輯、利潤試算 |
| `上架與通路.md` | 拍照/文案/上架平台、庫存管理、出貨流程 |
| `場地租借.md` | 拍攝/業配空間定價（NT$3,000-8,000/session）、檔期管理、預約流程 |

> 目前為空架構，等實際配方/成本資料輸入後，`/整理` 會照上表分流，或手動貼進對應子檔。

## 配方記錄建議格式
```
## 蠟燭名稱 / 建立日期
### 香精配方（品牌+比例）
### 蠟種與比例（大豆蠟/棕櫚蠟/蜂蠟…）
### 燃燒測試（香擴散度/燃燒時間/隧道效應有無）
### 成本（單支材料成本 NT$）
### 調香心得 / 下次調整
```
