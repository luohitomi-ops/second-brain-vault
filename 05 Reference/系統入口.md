## 自訂指令

### 大腦指令（Claude）
- `/每日校準` — 早上掃全局：卡關、今日焦點、今日發文、待回填、里程碑
- `/每週復盤` — 跨區週檢
- `/更新中樞` — 討論結論寫回中樞
- `/新工作流 名稱` — 建新節點並串連
- `/品牌深聊 [主題]` — 接續品牌顧問深挖，結論寫回品牌中樞（讓大腦長肉，不用重打背景）
- `/投資深聊 [主題]` — 接續投資顧問深挖（亞翔/房屋基金/00990A…），寫回投資中樞
- `/生活深聊 [主題]` — 接續醫美/技能/自律深挖，寫回對應分支
- ＊深聊預設 Sonnet；要深度思考才在前面加「深思」用 Opus

### 發文指令（Claude）
- `/發文日常` — 計畫表＋待回填＋F公式建議（三合一）
- `/爆款優化` — 分析戰績 → 更新 [[爆款路線分析]]
- `/整理Inbox` — 碎片靈感分類分流
- `/生成Brief` — 手動生成今日熱門話題摘要
- `/social-post` — 學語氣依 F1-F27 公式發文

### 投資指令（Claude）
- `/庫存健檢 [股]` — 召喚籌碼＋Minervini 逐支體檢，寫回 [[波段庫存]]

### Discord 免額度指令（bot 跑，不耗 Claude）
- `/brief` — Gemini 2.5-flash 生成今日 Brief
- `/整理` — Gemini 批次整理 Inbox 碎片 → 自動分類分流（發文素材/股票/醫美/技能/自律），0 Claude
- `/健檢` — 給獺金幣**程式**快照健檢（≠ /庫存健檢 個股體檢）
- 在 #快速捕獲 / #自律 直接打字 → 自動存檔（出現 ✅）

## 每天自動（bot 排程，免 Claude 額度）
- **02:00** 自動生成 Brief → 貼 #總控（Gemini）
- **平日 08:45** 自動股票健檢 → 貼 #總控（直連 stock-radar /api/health）

## 省額度機制（重要）
- **預設 Sonnet**：bot 每則訊息預設用 Sonnet（約 Opus 1/5 成本）
- **要 Opus 深思**：訊息開頭打「深思」或「/opus」才用 Opus（顧問深談、重大決策才需要）
- **純捕獲 0 額度**：#快速捕獲、#自律 直接打字 → 自動存檔，完全不經 Claude
- **生成/整理走 Gemini**：Brief（gemini-2.5-flash）、未來整理碎片走 Gemini（免費額度，批次處理）
- 分層原則：捕獲(免費) → 整理/生成(Gemini) → 例行(Sonnet) → 深度(Opus 只留這個)

## 頻道使用規則
- 頻道＝腦區：在哪個頻道發訊息就載入該區深背景（CLAUDE.md）
- 連續對話有記憶（同頻道）；`/reset` 或「重來」清空該頻道記憶
- #快速捕獲 / #自律：非指令訊息直接存檔，免額度即時
- 🧠 兩種記憶：聊天記憶只在當頻道暫存；要永久＋跨頻道看到 → 叫它「寫進對應中樞檔」（見記錄路由）
