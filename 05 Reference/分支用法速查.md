# 各分支怎麼用（速查）

> 每個區塊「怎麼輸入 / 怎麼回顧」。不確定丟哪 → 一律先丟 #快速捕獲，`/整理` 會分流。

## 通用法則
- **輸入(便宜)**：碎片丟 #快速捕獲 或 #自律（0 額度）→ `/整理`（Gemini）自動分流
- **回顧/深挖(才用Claude)**：`/X深聊` 接續該區，結論寫回中樞
- **看(零額度)**：Obsidian 直接讀

## 三個分支的「節奏」差異（最常搞混）
- **自律** = 每天打卡（高頻、#自律）
- **醫美** = 隨手累積心得（不定期、自動歸檔＋變素材）
- **技能** = 願望池 + 定期排序（低頻、/生活深聊 砍到一項）
