# 課程筆記庫

> **我自己上過的跨領域課程**的筆記整理（演員課／NLP／表達／溝通等）。
> **按應用場景分類，不按課程名稱分**——因為同一個技巧可能來自不同課，重點是「什麼情境用得到」。

## ⚠️ 這裡只放「我自己的技能學習」

跟兩種東西要分清楚（`/整理` 已設好分流規則，不會混）：
- **主持技巧課**（我上的、跟主持工作直接相關）→ 放 [[主持資料庫/03_資料庫/README|主持資料庫/03_資料庫]]，不放這裡
- **客戶/合作廠商給的產品資料**（如 ploom 產品教育內容，是為了主持他們活動用的，不是我的技能）→ 放 [[主持資料庫/README|主持資料庫]] 的事前準備區，**這不是我的課程筆記**

## 結構
`/整理` 目前把所有跨領域課程筆記存成獨立檔案放在這個資料夾根目錄（檔名：`課程筆記_{標題}_{日期}.md`），不分子資料夾——按應用場景分類（表達與台風／心理與溝通）等量多了再視情況手動整理。

## 課程筆記建議格式
```
## 課程名稱 / 講師 / 上課日期
### 核心觀念（這堂課最重要的 2-3 個 takeaway）
### 可直接套用的技巧
### 我的實戰驗證（哪個場合用過、效果如何 — 沒用過先留空）
```
