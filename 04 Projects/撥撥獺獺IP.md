# 撥撥獺獺 IP 宇宙

狀態：進行中｜類型：IP 開發 / 動畫 / 影片

## 角色版本
- 插畫版 / 動畫版 / 寫實版

## 角色設定（2026-07-04 確立）

### 撥撥
- 一隻**公的喜馬拉雅土撥鼠（旱獺）**
- 跟獺獺是同居情侶
- 特色裝備：黑色電競耳機、常背黑色包、有一台白色的車
- 個性：表情呆萌、會突然發呆、因土撥鼠大叫迷因愛突然大叫
- 興趣：打遊戲、很愛吃、很會睡；喜歡帶獺獺到處去冒險

### 獺獺
- 一隻**母的小爪水獺**
- 招牌 W 微笑嘴、背檸檬造型斜背包
- 個性：可愛但衝動急躁、家事白痴、不會做飯
- 興趣：喜歡到處探店、愛美愛打扮

### 姆妞
- 撥撥跟獺獺的**寵物玳瑁貓**
- 個性：乖巧可愛、有好奇心、平時很乖
- 做錯事會心虛喵喵叫或躲起來
- 把撥撥獺獺當成父母，他們在做什麼都要湊一腳

## 內容策略（2026-07-04 確立）

### 核心定位
**一個「情侶+貓」的日常宇宙** — 天然覆蓋情侶互動、貓咪、食物探店三類低略過率內容

### 五條主力內容線

**① 情侶吵架但沒有辦法（最強）**
- 撥撥呆萌 vs 獺獺衝動急躁 = 天然喜劇張力
- 情境：獺獺說要做飯→後果；撥撥打遊戲無視獺獺→獺獺爆炸；獺獺選餐廳→撥撥車上睡著；吵完姆妞湊過來兩人停戰
- 鉤子公式：前1秒是衝突發生瞬間，不鋪墊

**② 獺獺探店 vlog（食物，略過率最低）**
- 撥撥開白色車抵達 → 獺獺衝動點餐 → 好吃W嘴笑到極限 / 踩雷獺獺爆炸撥撥呆
- 同時打：情侶出遊、食物、反應meme 三個流量池

**③ 土撥鼠大叫 meme（低成本高傳播）**
- 成熟迷因格式有現成配樂受眾
- 模板：獺獺說某事→撥撥發呆3秒→大叫；姆妞做什麼→撥撥大叫
- 30秒內，前1秒撥撥呆臉準備發呆；可批量做一個模板5-10個情境

**④ 姆妞做壞事（貓咪流量）**
- 姆妞打翻什麼→自己心虛躲起來→撥撥獺獺到處找→找到縮成一團→被抱起來沒事了
- 真實貓行為 × 認父母人設 = 普世高傳播

**⑤ 撥撥的車×出發去冒險（情侶旅遊感）**
- 白色車 = 場景切換道具
- 出門前：獺獺盛裝打扮、撥撥還在睡 → 車上：撥撥開車獺獺選歌選錯 → 抵達：反應差異
- 番外：姆妞獨自在家那段時間

### 要避免的
| 避免 | 原因 |
|------|------|
| 遊戲相關內容 | 已驗證略過率 84.8%，IP受眾太窄 |
| 三個角色同時都在動 | 畫面太滿，焦點散 |
| 無衝突/無反應的純展示 | 沒有情緒落點，前1秒沒有吸力 |

### 第一批主題（驗證哪條線最強）
1. 「獺獺說要做飯」（情侶吵架線）
2. 「撥撥突然大叫」一個情境（meme線）
3. 「姆妞打翻什麼後躲起來」（貓咪線）
